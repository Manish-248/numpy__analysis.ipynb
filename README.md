# 📊 NumPy Fundamentals & Data Analysis Toolkit

## 🚀 Overview

This repository contains a comprehensive implementation of **NumPy fundamentals and data analysis techniques** using Python.
The project demonstrates core numerical computing concepts along with practical examples used in data science workflows.

---

## 🎯 Objectives

* Understand NumPy array structures and operations
* Perform data manipulation using indexing and slicing
* Apply vectorization for efficient computation
* Use broadcasting for operations on different shapes
* Generate and analyze random data

---

## 📚 Topics Covered

### 🔹 NumPy Fundamentals

* Array creation (1D, 2D, ndarrays)
* Data types and structure (`dtype`, `shape`, `ndim`)
* Reshaping and transformations

### 🔹 Indexing & Slicing

* Basic indexing
* Fancy indexing
* Boolean masking

### 🔹 Array Operations

* Element-wise operations
* Mathematical functions
* Aggregations (mean, sum, std)

### 🔹 Vectorization

* Loop-free computations
* Performance optimization

### 🔹 Broadcasting

* Operations on arrays with different shapes
* Implicit expansion rules

### 🔹 Random Number Generation

* Uniform and normal distributions
* Reproducibility using seeds
